# Changelog

All notable changes to Real Black are documented here.

## 1.0.0 - 2026-07-05

- First stable open-source release.
- Includes the Real Black CLion UI theme, custom C++ editor color scheme, C++ keyword annotator, documentation, screenshot, release checklist, and MIT license.
- Uses the Marketplace-ready plugin ID `dev.nazimaraz.realblack`.
- Uses `RealBlack` resource names and Real Black naming across metadata, documentation, and the C++ color specimen.
- Declares CLion module compatibility explicitly for JetBrains Marketplace.

## 0.17.0 - 2026-07-05

- Added the first README screenshot.
- Renamed the timestamped screenshot to `docs/screenshots/cpp-editor.png`.
- Updated screenshot documentation.

## 0.16.0 - 2026-07-05

- Updated plugin author/vendor attribution to Nazım Araz.
- Updated theme author metadata and MIT copyright attribution.

## 0.15.0 - 2026-07-05

- Removed underline effects from identifier occurrence highlights.
- Kept the stronger read/write occurrence background colors.

## 0.14.0 - 2026-07-05

- Brightened identifier occurrence highlights in the editor.
- Added a separate write-occurrence highlight for identifiers under the caret.
- Increased text/search occurrence highlight visibility.

## 0.13.0 - 2026-07-05

- Added GitHub-ready open source documentation.
- Added MIT license.
- Added contributing notes, release checklist, and screenshot capture checklist.
- Expanded the README with installation, development, release, and project layout details.

## 0.12.0 - 2026-07-05

- Added README note that Real Black is fully vibe coded.

## 0.11.0 - 2026-07-05

- Removed the custom rainbow-depth bracket, parenthesis, and brace annotator.
- Removed custom rainbow delimiter scheme keys.
- Restored normal CLion punctuation coloring for delimiters.

## 0.10.0 - 2026-07-04

- Brightened modified file name colors.
- Added bright green colors for added and unversioned file names.

## 0.9.0 - 2026-07-04

- Set modified file names to an Islands Dark-style link blue.
- Added VCS status wiring for modified files.

## 0.8.0 - 2026-07-04

- Made the UI more black again.
- Matched excluded and ignored directory colors more closely to Islands Dark.
- Increased project tree/list row sizing toward Islands Dark.

## 0.7.0 - 2026-07-04

- Refined the UI shell toward Islands Dark with layered charcoal panels.
- Calmed selections and borders.
- Fixed invisible plugin install button text.

## 0.6.0 - 2026-07-04

- Renamed the plugin and theme to Real Black.
- Updated package naming to `real-black`.
- Preserved the editor color scheme while reshaping the IDE UI shell.

## 0.5.0 - 2026-07-04

- Added custom rainbow-depth delimiter coloring for nested parentheses, brackets, and braces.

## 0.4.0 - 2026-07-04

- Added a C++ specimen file for checking theme color coverage.
- Included broad C++ syntax samples for keywords, templates, types, operators, and modern language features.

## 0.3.0 - 2026-07-04

- Improved out-of-class definition coloring for patterns such as `auto ChartRenderer::draw() const -> void`.
- Added distinct coloring for owner types, callable names, and trailing return types.

## 0.2.0 - 2026-07-04

- Added the custom C++ annotator.
- Split keyword coloring for `namespace`, `auto`, `const`, `constexpr`, `template`, built-in types, and related tokens.

## 0.1.0 - 2026-07-04

- Initial black CLion theme prototype.
